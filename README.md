# 🎵 ChordsPad

Εργαλείο γρήγορης εισαγωγής συγχορδιών σε αρχεία κειμένου, με ταυτόχρονη ακρόαση του ήχου κάθε συγχορδίας.

---

## 📥 Εγκατάσταση

Δεν χρειάζεται εγκατάσταση. Απλώς:

1. Κατέβασε το **[ChordsPad.exe](https://github.com/perdix1975/ChordsPad/releases/download/v1.0/ChordsPad.exe)**
2. Τρέξε το αρχείο

---

## 🖥️ Πώς λειτουργεί

Το ChordsPad είναι ένα μικρό παράθυρο που μένει πάντα στην κορυφή της οθόνης (always on top). Δουλεύεις στον επεξεργαστή κειμένου σου (Notepad++, Word, κλπ) και χρησιμοποιείς το ChordsPad δίπλα για να εισάγεις συγχορδίες με ένα κλικ.

---

## 🎼 Μορφή ChordPro

Το ChordsPad εισάγει τις συγχορδίες σε μορφή **[ChordPro](https://www.chordpro.org/)** — το παγκόσμιο πρότυπο για αρχεία στίχων με συγχορδίες. Κάθε συγχορδία εισάγεται μέσα σε αγκύλες, π.χ. `[Am]`, `[C#m]`, `[G]`, και τοποθετείται μέσα στο κείμενο των στίχων ακριβώς στο σημείο που αλλάζει η συγχορδία:

```
[Am]Πάνω στη μαύρη [Dm]θάλασσα
```

Τα metadata (τίτλος, στιχουργός, μουσικός κλπ) εισάγονται επίσης σε μορφή ChordPro:

```
{t:Τίτλος Τραγουδιού}
{composer:Όνομα Μουσικού}
{artist:Όνομα Καλλιτέχνη}
{key:Am}
```

---

## 🎹 Στήλες κουμπιών

| Στήλη | Λειτουργία |
|-------|------------|
| **Play Chord** | Παίζει τον ήχο της συγχορδίας χωρίς να εισάγει κείμενο |
| **Insert Ch** | Εισάγει τη συγχορδία στο ενεργό παράθυρο (π.χ. `[Am]`) και παίζει τον ήχο |

---

## 📝 Πεδία Metadata

Στην κορυφή υπάρχουν πεδία για τα στοιχεία του τραγουδιού:

| Πεδίο | Περιεχόμενο |
|-------|------------|
| **Τ** | Τίτλος τραγουδιού |
| **Σ** | Στιχουργός |
| **Μ** | Μουσικός |
| **Κ** | Καλλιτέχνης |
| **Key** | Τονικότητα |

Πατώντας **Insert** εισάγονται αυτόματα όλα τα metadata στην αρχή του αρχείου σε μορφή ChordPro.

---

## 🎨 Κουμπιά χρωμάτων

- **Blue / Green / Red**: Εισάγουν ετικέτες χρώματος (`<span color=blue>` κλπ) για μορφοποίηση κειμένου.

---

## 🎼 Διαθέσιμες συγχορδίες

A, Am, Bb, Bbm, B, Bm, C, Cm, C#, C#m, D, Dm, D#, D#m, E, Em, F, Fm, F#, F#m, G, Gm, G#, G#m, Adim, Ddim, Edim

---

## ⌨️ Ειδικά κουμπιά

| Κουμπί | Λειτουργία |
|--------|------------|
| **S. C.** | Εισάγει "σόλο κουπλέ" |
| **S. R.** | Εισάγει "σόλο ρεφρέν" |
| **Undo** | Αναίρεση τελευταίας πράξης |
| **δις** | Εισάγει "δις" |
